# 基于 AI 开发的基于双索引架构的中文语义检索引擎

#### 介绍
基于倒排索引和语义向量双索引架构的中文语义检索引擎，支持从Gitee同步仓库数据并提供全文搜索功能。系统结合传统倒排索引搜索与基于text2vec-base-chinese的语义向量搜索，实现更精准的检索效果。

#### 项目技术栈

- 程序语言：Java、Python、css、html
- Java版本: 17
- 框架：Spring Boot 3.3.0、Node
- 数据库：MySQL
- 索引：自定义倒排索引
- 语义：text2vec-base-chinese大模型
- 分词工具：Jieba分词
- 项目管理：Maven 4.0.0
